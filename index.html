P.94  5 生活習慣病とその予防

1 生活習慣病とは

生活習慣病

とは、

生活習慣

がその

発症

や

進行

に関係する病気のことです。

高血圧

や

糖尿病

のほか、日本人の死因の多くを占める、

がん

、

心臓病

、

脳卒中

などの多くは、生活習慣と関係が深いことがわかっています。問題となる生活習慣には、

食生活の乱れ

、

運動や休養の不足

、

喫煙

、

過度の飲酒

などがあります。生活習慣は

子供

の頃にその基本が作られます。一度身についた

生活習慣

を変えるのは難しく、長年に渡って続いた結果、

生活習慣病

が発症するのです。
また、近年では、子供のうちから生活習慣病になる例も増加してきています。

■ 循環器の病気

動物性脂肪

の

とりすぎ

や

運動不足

などは、

動脈硬化

に繋がります。

動脈硬化

は、

血管

の壁に

コレステルール

などの

脂肪

がたまり、血管が

硬く

、

もろく

なった状態です。また、

塩分

のとりすぎや

ストレス

などは、

高血圧

に繋がります。

高血圧

は、

動脈

にかかる

圧力

が異常に

高く

なった状態です。

動脈硬化

や

高血圧

は、

心臓病

や

脳卒中

を引き起こすもとになります。

心臓病

には、心臓の

筋肉

に血液を供給する血管が

狭く

なっておこる

狭心症

と、血管が詰まって起こる

心筋梗塞

があります。

脳卒中

には、脳に血液を供給する血管が詰まって起こる

脳梗塞

と、血管が

破れて

起こる

脳出血

があります。
************************************************************************

コレステロールには、健康に害を及ぼしやすい

LDLコレステロール

と、血管の壁に脂肪がたまるのを防ぎ、健康に良い働きをする

HDLコレステロール

がある。

LDLコレステロール

が多すぎたり、

HDLコレステロール

が少なすぎたりするなどの状態を

脂質異常症

という。
************************************************************************

高血圧症

、

脂質異常症

、

糖尿病

は、「

サイレントキラー（silent killer）

」とも呼ばれています。自覚症状が殆どないまま、血管に影響を及ぼし、

動脈硬化

を進行させるからです。


************************************************************************

■ がん

喫煙

や

塩分

のとりすぎ、

野菜

・

くだもの

の不足などは、

がん

に繋がります。

がん

は、正常な

細胞

の

遺伝子

が傷ついて

がん細胞

に変化し、その

がん細胞

が無秩序に

増殖

して

器官

の働きを侵してしまう病気です。

■ 糖尿病

エネルギー

のとりすぎや

運動不足

、

肥満

などは、

糖尿病

に繋がります。

糖尿病

は、血液に含まれる

ブドウ糖

の量が異常に

多く

なる病気です。

糖尿病

になると、血管に

負担

がかかるため、

心臓病

や

脳卒中

、

腎臓

や

目

、

血管

、

神経

の

障害

など、様々な病気が起こるようになります。

2 生活習慣病の予防

生活習慣病

は、

生活習慣

をできるだけ望ましいものにして健康を

増進

し、

発病

そのものを

予防

する事が重要です。また、

定期的

に

検査

を受けることも有効です。

生活習慣病

は

初期

にはほとんど

自覚症状

がなく、じわじわと進行するからです。

検査

によって

早期発見

・

早期治療

をし、

生活習慣

を

改善

できれば、その多くは進行を食い止めることができるのです。
現代の社会は、

食生活の乱れ

や

運動不足

、

ストレス

が起こりやすくなっています。

健康的

な

生活習慣

づくりには、個人の

意識的

な取り組みとともに、個人の

取り組み

を

支援

する

社会的環境

の

整備

も必要です。



■ メタボリックシンドローム

内蔵やその周辺に溜まった脂肪（

内臓脂肪

）が生活習慣病の発症に深く関わっていることがわかっています。この

内臓脂肪

の蓄積に加えて、

高血圧

、

高血糖

、

脂質異常

のうち、二つ以上が当てはまる状態を

メタボリックシンドローム

（

内臓脂肪症候群

）と呼びます。

メタボリックシンドローム

になると、生活習慣病になる危険性が急激に高くなります。

メタボリックシンドローム

かどうかは、

腹囲（へそ周り）

、

血圧

、

血液検査

などから診断します。
************************************************************************

「

一無二少三多

」という考え方があります。

一無

は

無煙

、

禁煙

、

二少

は

少食

、

少酒

、

三多

は

体

を多く動かし（

多動

）、しっかり

休養

をとる（

多休

）、多くの

人、事、物

に接する生活（

多接

）です。
************************************************************************

P.98  がん

がんは、日本人の約

二

人に一人がかかり、死因の約

30

%を占める病気です。

部位

による違いや

男女

による違いがあります。
がんは、

遺伝子

がきずつくことによって起こる病気であり、

感染

する病気ではありません。また、国民の多くががんを患うことから、がんは

国民病

とも呼ばれています。

主ながんによる死亡の部位別割合では、男性では

肺

が最も多く、次に

胃

や

大腸

が高い割合を占めています。女性では、

大腸

が最も多く、次に

肺

や

胃

が高い割合を占めています。

P.99  アレルギー

日本人の

三

人に一人は、何らかのアレルギーがあるといわれています。アレルギーは

アレルギー物質

（

アレルゲン

）、

体質

（

素因

）、

生活習慣

や

ストレス

などの要因が関係して起こります。

アレルギー物質

の多くは、本来、体に

害

を及ぼすものではありませんが、

免疫

（体の

抵抗力

）が

過剰

に働いて、様々な症状を起こしてしまうのです。

アレルギー物質（アレルゲン）の例
吸い込むもの : 

花粉

ダニ

ほこり

カビ

など
食べるもの : 

卵

牛乳

小麦

そば

落花生（ピーナッツ）

大豆

甲殻類

（えび、かになど）

くだもの

（もも、メロンなど）

など
体に直接入るもの : 

注射

や

点滴

などで体内に入る

薬

スズメバチ

の

毒

など
皮膚にふれる物 : 

金属

化学物質

など
